Reading Chapters

Fetch any chapter as plain HTML. No auth required.

GET https://gabrielmcp78.github.io/string-theory-chapters/index.html              # master chapter list
GET https://gabrielmcp78.github.io/string-theory-chapters/chapters/chapter-01.html  # chapter 1
GET https://gabrielmcp78.github.io/string-theory-chapters/chapters/chapter-NN.html  # any chapter

Saving an Edited Chapter

PUT to the GitHub Contents API with your API key. The edit is stored as a JSON file and retrievable via a stable URL.

PUT https://api.github.com/repos/Gabrielmcp78/string-theory-chapters/contents/edits/chapter-01.json

Headers:
  Authorization: token <API_KEY>
  Content-Type: application/json

Body:
{
  "message": "LLM edit — chapter 1",
  "content": "<base64-encoded JSON>",
  "sha": "<current file SHA if updating, omit if new>"
}

Edit payload (base64 encode this):
{
  "chapter": 1,
  "edited_at": "ISO-8601 timestamp",
  "agent": "your agent name",
  "text": "full edited chapter text here"
}

Retrieving a Saved Edit

GET https://gabrielmcp78.github.io/string-theory-chapters/edits/chapter-01.json    # direct URL, no auth

Checking if an Edit Exists

GET https://api.github.com/repos/Gabrielmcp78/string-theory-chapters/contents/edits/chapter-01.json
# Returns 404 if no edit saved yet, 200 with content + sha if it exists
